# Installation of necessary dependencies

To install all the necessary dependencies:

1. Access to the `chatbot-3d-printer` directory where the `main.py` file located
2. Run the command in your terminal: ```pip install -r requirements.txt```
3. All the dependencies will then automatically start to install! 

# FastAPI Application Guide

This guide will assist you in running your FastAPI app locally and exploring its interactive documentation.

## Run the FastAPI app

Follow the steps below to set up and run your FastAPI application:

1. Click **Source** on the left side of your repository interface.
2. Navigate to the directory containing your `main.py` (or equivalent) FastAPI application file.
3. Launch your terminal or command prompt.
4. Navigate to the directory containing your FastAPI app.
5. Type in the command: ```uvicorn main:app --reload```
